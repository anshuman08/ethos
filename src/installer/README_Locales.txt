Ethosmiracle  is delivered with support for 15 languages.

Full translations provided by Ethosmiracle include:
English UK - the default language set.
English - USA
French - European
German - UI mostly complete. Reports approx' 75%
Spanish - UI mostly complete. Reports approx' 75%

Not all locales are completely translated.

The locale files shipped with this version of Ethosmiracle are the 
latest at the date of this release.

Latest locales can be found here: https://Ethosmiracle.com

Application Locales (Default):
beans_messages.properties is the Java generic message file
data_messages.properties is relevant to the data interface 
pos_messages.properties is the MAIN application label and Messages file

Reports:
Each report has a .properties file associated with it.
You will find the files in the Ethosmiracle installation folder in the
Ethosmiracle instalation sub-folder \reports\com\openbravo\reports

HOW TO CHANGE YOUR LANGUAGE
All languages are now compiled into the Ethosmiracle executable.
Any required locale file changes will require the project to be recompiled.

Set the Ethosmiracle Configuration>Locale to your required language
